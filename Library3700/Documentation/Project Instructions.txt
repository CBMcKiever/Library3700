Library Information Report:

This is a short report for assisting in running the Library Information System Locally.

(1) Install Microsoft Visual Studio

	- If you already have Microsoft Visual Studio installed, continue to step 2.

	- If not you can install Microsfot Visual Studio from this link: https://www.visualstudio.com/downloads

(2) Install the ASP.NET and Web Development workload

	- This step can be handled during the install by selecting the ASP.NET and We Development workload when
	  Visual Studio gives you the option during install. 

	- If you already have Visual Studio installed, but not the ASP.NET and Web Development workload, you can
          download it using the Visual Studio Installer.

(3) Download Project from Canvas Submission

(4) Open the Project in Visual Studio

(5) Run the Project
